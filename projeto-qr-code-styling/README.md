# 🎬 Gerador de QR Code Customizável

## 📝 Descrição do Projeto

Este projeto é uma ferramenta avançada e intuitiva para a geração de QR Codes totalmente personalizados. Desenvolvido com foco em flexibilidade e design, o sistema permite que usuários criem códigos únicos ajustando desde a geometria dos pontos até a inclusão de logotipos centrais e gradientes complexos.

O objetivo principal é oferecer uma solução rápida para empresas e indivíduos que desejam manter a identidade visual de sua marca mesmo em elementos técnicos, mitigando a aparência genérica de QRs convencionais e aumentando o engajamento visual.

## 🚀 Tecnologias Utilizadas

*   **Linguagem:** TypeScript
*   **Framework:** React 18
*   **Estilização:** Tailwind CSS & Framer Motion
*   **Motor de Renderização:** QR-Code-Styling
*   **Ícones:** Lucide React

## 📊 Recursos e Aprendizados

O projeto foca na manipulação dinâmica do DOM para renderização de SVG/Canvas, garantindo que as alterações do usuário sejam refletidas instantaneamente.

*   **Customização Granular:** Implementação de controles para 6 estilos de pontos, 4 molduras de cantos e preenchimentos internos.
*   **Manipulação de Imagens:** Suporte para upload de logos com controle de transparência de fundo e margens de proteção.
*   **Exportação Multiformato:** Sistema de download otimizado para PNG, JPEG e SVG (vetorial).
*   **UX Responsiva:** Layout adaptável com sistema de abas colapsáveis para facilitar a edição em dispositivos móveis.
