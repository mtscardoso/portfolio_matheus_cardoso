# 🎨 Qryo: Identidade Visual em cada Pixel

## 📝 Descrição do Projeto

O **Qryo** é uma plataforma premium de geração e personalização de QR Codes que redefine a forma como marcas e usuários interagem com códigos bidimensionais. O projeto nasceu da necessidade de transformar QR Codes puramente funcionais em elementos artísticos e integrados à identidade visual de uma marca, mitigando a estética genérica e "industrial" do QR Code padrão.

Desenvolvido com foco em **experiência de usuário (UX) e privacidade**, o sistema utiliza técnicas de neumorfismo para uma interface táctil e moderna, permitindo a criação de códigos únicos com padrões personalizados, logotipos integrados e gradientes sofisticados, tudo processado localmente no navegador para garantir a segurança dos dados.

## 🚀 Tecnologias Utilizadas

* **Framework:** React 19 + TypeScript

* **Estilização:** Tailwind CSS v4 + Motion (Animações)

* **Backend & Infra:** Firebase (Authentication, Firestore, Remote Config)

* **Processamento Local:** QR-Code-Styling & JSZip

## 📊 Diferenciais e Funcionalidades

O Qryo entrega uma solução completa que vai além da simples geração de links.

* **Personalização Granular:** Controle total sobre padrões de pontos, estilos de cantos (frames e eyeballs) e paleta de cores.

* **Modo Turbo (Batch Generation):** Capacidade de gerar centenas de QR Codes simultaneamente a partir de uma lista de URLs, entregando um arquivo `.ZIP` em segundos.

* **Persistência Local (Presets):** Sistema de favoritos que permite salvar designs complexos diretamente no `localStorage` do usuário, respeitando a privacidade total.


## 🔧 Como Executar

1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Inicie o ambiente de desenvolvimento: `npm run dev`.
