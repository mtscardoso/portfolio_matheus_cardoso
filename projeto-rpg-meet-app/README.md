## 💡 Proposta de Valor

Organizar sessões de RPG remotas pode ser um desafio logístico. O **RPG Meet** resolve esse problema ao centralizar a comunicação em uma interface otimizada, permitindo que a única preocupação do grupo seja a rolagem dos dados. 

Diferente de apps de reuniões genéricos, este projeto foi estruturado para ser leve, direto ao ponto e integrado a ferramentas que fazem parte do ecossistema de quem joga online.

### Diferenciais:
*   **Foco na Agilidade:** Crie ou entre em salas em segundos.
*   **Estrutura Otimizada:** Código refinado via **Manus AI** para máxima performance em dispositivos Android.
*   **Privacidade e Estabilidade:** Utiliza a infraestrutura robusta do Jitsi Meet.

---

## 🚀 Funcionalidades

- [x] **Integração Jitsi Meet:** Criação e entrada em salas de videoconferência reais.
- [x] **Interface Intuitiva:** Design focado na experiência do usuário (UX).
- [x] **Multiplataforma:** Simulação funcional via web e pronto para APK Android.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no desenvolvimento mobile e IA:

*   **Android (Kotlin/Java):** Base do desenvolvimento nativo.
*   **Jitsi Meet SDK:** Motor de comunicação para chamadas de vídeo e áudio.
*   **Manus AI:** Utilizada para a otimização da arquitetura do código e geração de interface.
*   **Markdown:** Para documentação técnica.
