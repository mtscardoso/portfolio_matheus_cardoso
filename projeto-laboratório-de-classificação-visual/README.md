# 👁️ Laboratório de Classificação Visual e Ética em IA

## 📝 Descrição do Projeto
Este projeto explora o treinamento de modelos de aprendizado de máquina utilizando a ferramenta **Teachable Machine**. O objetivo central foi analisar como o desenvolvimento de modelos de classificação visual pode ser influenciado por vieses humanos e as implicações éticas de automatizar julgamentos subjetivos.

O experimento consistiu em treinar uma IA para classificar imagens de mãos entre categorias binárias e subjetivas ("Mão Bonita" vs "Mão Feia"), servindo como um estudo de caso sobre a responsabilidade técnica e social no desenvolvimento de algoritmos.

## 🚀 Tecnologias Utilizadas
* **Ferramenta:** Teachable Machine (Google)
* **Técnica:** Machine Learning (Classificação de Imagem)
* **Hardware:** Webcam para captura de dados em tempo real.

## 📊 Resultados e Aprendizados
O experimento proporcionou reflexões críticas sobre a natureza dos dados e a lógica dos algoritmos:

* **Mecanismo de Viés:** Identificou-se que a seleção de dados foi baseada em estereótipos. Ao fornecer um contexto fechado, a máquina é induzida a "enxergar o que o desenvolvedor quer", o que torna o ensino de conceitos complexos (como beleza) através de lógica binária (0 ou 1) um equívoco técnico.
* **Consequência Social:** A incapacidade do programa em lidar com a complexidade humana pode resultar em negligência com o indivíduo. Tratar temas sensíveis de forma simplificada pode acarretar consequências psicológicas negativas para as pessoas classificadas pelo sistema.
* **Ação Mitigadora:** Para melhorar o modelo, é necessário adicionar vertentes baseadas na complexidade do assunto. Uma amplitude descritiva maior permite que a máquina obtenha respostas mais precisas e resultados menos estereotipados.
