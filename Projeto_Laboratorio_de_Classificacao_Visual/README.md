# 🤖 Análise de Viés em Sistemas de Visão Computacional
 
## 📝 Descrição do Projeto
Este projeto explora os perigos éticos e técnicos de reduzir conceitos subjetivos e complexos (como o "conceito de bonito") a classificações binárias (0 ou 1). Utilizando a ferramenta **Teachable Machine**, o experimento demonstra como o treinamento de modelos baseado em contextos fechados e conjuntos de dados limitados pode replicar e amplificar estereótipos humanos.
 
Desenvolvido como uma **análise crítica de Algoritmos e Ética**, o sistema foi treinado para classificar mãos entre "Mão Bonita" e "Mão Feia", servindo como um caso de estudo sobre como a IA pode ser "negligente com o indivíduo" ao ignorar a diversidade e a complexidade da realidade humana.

## 🚀 Tecnologias Utilizadas
* **Plataforma:** Google Teachable Machine
* **Técnica:** Classificação de Imagem (Image Recognition)
* **Conceitos:** Aprendizado de Máquina Supervisionado, Ética em IA, Bioética Digital
 
## 📊 Resultados e Aprendizados
A análise revelou que sistemas de IA frequentemente "enxergam o que nós queremos", sem filtrar se a base de dados é correta ou justa do ponto de vista social.
* **Mecanismo de Viés:** Identificou-se que a seleção binária é totalmente baseada em estereótipos, falhando em determinar conceitos amplos em contextos restritos.
* **Impacto Social:** O estudo destacou como classificações simplistas podem acarretar consequências psicológicas negativas e negligência com a identidade do usuário.
* **Ação Mitigadora:** Proposta de inclusão de "mais vertentes" e "amplitude descritiva" para que a máquina obtenha respostas mais precisas e menos estereotipadas na medida do possível.

## 🔧 Como Executar
1. Acesse o [Teachable Machine](https://teachablemachine.withgoogle.com/).
2. Carregue classes de imagens que representem conceitos subjetivamente opostos (ex: Bonito vs. Feio).
3. Observe como a máquina associa padrões visuais aos rótulos (labels) fornecidos sem critério ético.
4. Aplique as ações mitigadoras sugeridas (aumento de complexidade e descrição) para reduzir o impacto do viés.
