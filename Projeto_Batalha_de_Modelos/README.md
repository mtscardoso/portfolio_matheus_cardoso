# ⚔️ Batalha de Modelos & Engenharia de Prompt (XML)
 
## 📝 Descrição do Projeto
Este projeto consiste em um experimento de **Engenharia de Prompt Estruturado**, utilizando a sintaxe XML para guiar diferentes modelos de Inteligência Artificial na geração de código front-end (Single Page HTML5 com CSS3). O objetivo foi avaliar a capacidade de compreensão lógica, criatividade e precisão técnica de diversos LLMs ao seguirem um protocolo rigoroso de design e desenvolvimento.
 
O sistema de prompt foi testado em uma "batalha" entre os modelos: **ChatGPT, Gemini, DeepSeek, Qwen, Grok, Maritaca e Claude**, comparando desde a verbosidade (consumo de tokens) até a funcionalidade real do código gerado.

## 🚀 Tecnologias Utilizadas
* **Engenharia de Prompt:** Estruturação via XML (Context-Specific Prompting)
* **Modelos Avaliados:** Claude, DeepSeek, Gemini, GPT-4, Qwen, Grok e Maritaca
* **Linguagens de Saída:** HTML5, CSS3 e JavaScript (opcional por alguns modelos)
 
## 📊 Resultados e Aprendizados
O experimento revelou discrepâncias significativas entre a "compreensão" de estrutura e a "criatividade" dos modelos.
* **Liderança em Compreensão:** O **Claude** foi eleito o melhor em compreender a estrutura XML, entregando resultados criativos, únicos e altamente alinhados ao pedido.
* **Eficiência vs. Complexidade:** O **Qwen** demonstrou ser ideal para prototipagem rápida (apenas 350 tokens), enquanto o **Claude** utilizou mais de 6.000 tokens para gerar códigos extremamente expansivos.
* **Análise de Falhas:** Modelos como GPT e Maritaca apresentaram dificuldades em gerar sites funcionais ou omitiram informações cruciais (links quebrados e imagens ausentes).

## 🔧 Como Executar
1. Utilize o **Protocolo de Execução** em XML (fornecido na página 1 do relatório).
2. Insira o tema desejado na tag `<tema>` e a paleta de cores em `<paleta_cores>`.
3. Execute o prompt em uma ferramenta de IA (recomenda-se **Claude** para alta complexidade ou **Qwen** para rapidez).
4. Verifique a estimativa de tokens gerados conforme a métrica obrigatória do prompt.
